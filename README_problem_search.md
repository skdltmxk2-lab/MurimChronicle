# 편입수학 문제 검색 MVP

Streamlit 기반의 반자동 프로토타입입니다.

## 실행

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
streamlit run streamlit_app.py
```

Tesseract OCR을 쓰려면 Windows에 Tesseract 본체와 한국어/영어 언어팩이 설치되어 있어야 합니다. 설치하지 않아도 문제 텍스트를 직접 입력해서 저장하고 검색할 수 있습니다.

## OCR 설치

현재 앱의 OCR 버튼은 Windows용 Tesseract 실행 파일을 사용합니다. `pytesseract`는 파이썬 연결 패키지일 뿐이라 Tesseract 본체를 따로 설치해야 합니다.

1. UB Mannheim Tesseract Windows installer를 설치합니다.
2. 설치 옵션에서 Korean language data와 English language data를 포함합니다.
3. 기본 경로 `C:\Program Files\Tesseract-OCR\tesseract.exe`에 설치하면 앱이 자동으로 찾습니다.
4. 설치 후 Streamlit 앱을 껐다가 다시 실행합니다.

설치 전에도 문제 영역 캡처, 직접 텍스트 입력, 저장, 검색은 그대로 사용할 수 있습니다.

## 현재 기능

- PDF 업로드 후 페이지 이미지 변환
- 이미지 파일 업로드
- 앱 안의 붙여넣기 영역에서 Ctrl+V 이미지 입력
- Windows 클립보드 이미지 가져오기 버튼
- 올린 이미지 전체를 한 문제로 저장
- 필요할 때만 슬라이더로 일부 영역 저장
- 선택 영역 OCR 실행
- 현재 페이지의 문제 번호 기반 자동 분리/자동 저장
- PDF 전체에서 문제 번호를 감지해 일괄 자동 저장
- 이미지 파일 최대 10개를 각각 한 문제로 일괄 저장
- 업로드 전체 문제 개수 감지 및 문제별 단원/난이도/유사도 미리보기
- 문제/해설 후보를 구분해 해설은 별도 저장
- 검색 결과와 라이브러리에서 연결된 정답/풀이 보기
- 문제 단위 SQLite 저장
- 원본 파일명, 페이지, 문제 번호, bbox, crop 이미지 저장
- 단원/유형/풀이 태그/난이도 저장
- 문제 crop 이미지 기반 유사도 검색
- 텍스트 유사도와 수식 구조/풀이 태그 가중 검색
- 이미지 유사도와 개념/단원 유사도 가중치 조절 검색
- 검색 결과에서 문제 영역 crop 표시 및 원본 페이지 확인
- 업로드 문제의 과목/대단원/소단원/난이도/유형 태그 자동 추천
- Gemini/OpenAI API 키를 이용한 AI 이미지 분류 추천
- 과목 선택 후 해당 과목의 대단원/소단원만 선택
- 라이브러리에서 과목/대단원/소단원별 묶음 열람

## 이미지 유사도 검색

저장할 때 문제 crop 이미지에서 간단한 구조 feature를 만들어 DB에 저장합니다. 검색할 때 입력 문제 이미지를 붙여넣거나 업로드하면 이미지 유사도와 단원/풀이태그/텍스트 유사도를 섞어서 결과를 정렬합니다.

이미지 유사도는 같은 스캔본, 비슷한 수식 배치, 유사한 보기 구조를 찾는 데 유용하지만, 풀이법이 같은 문제를 완벽히 판단하지는 못합니다. 그래서 검색 화면에서 이미지 반영 비율을 조절할 수 있게 했습니다.

## AI 분류

사이드바의 `AI 분류` 영역에서 `Gemini` 또는 `OpenAI`를 선택하고 API 키를 입력한 뒤 저장 화면에서 `AI로 자동 분류`를 누르면, 문제 이미지를 직접 보고 과목/대단원/소단원/난이도/유형 태그를 추천합니다.

API 키는 현재 브라우저 세션에만 보관하며 코드나 DB에 저장하지 않습니다. AI 추천도 확정값이 아니므로 저장 전 사용자가 수정할 수 있습니다.

## 데이터 저장 위치

`problem_search_data/`

- `uploads/`: 업로드 원본
- `pages/`: PDF에서 렌더링한 페이지 이미지
- `crops/`: 문제별 캡처 이미지
- `solutions/`: 해설별 캡처 이미지
- `problems.sqlite3`: 문제 메타데이터
